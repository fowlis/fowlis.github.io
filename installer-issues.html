Common Installer Issues

Installer doesn't open
Installer doesn't open at all

a. Do a manual installation > guide
b. Try the 7-Zip method:

  1. Install 7-Zip or similar software.
  2. Right click the BetterDiscord-windows.exe, from the 7-Zip options, click "extract to 'BetterDiscord-Windows\'"
  3. Open the new BetterDiscord-Windows folder and run the BetterDiscord.exe

"Error opening file for writing"

a. Click "ignore" several times (genuinely, like 20-30 times)
b. Do a manual installation > guide

No Discord versions to select
Windows users:
  1. Unpin the Discord icon from your taskbar (if it's pinned)
  2. Quit or Uninstall Discord
  3. Reinstall Discord from the Discord site
  4. Try the installer again
Linux users:

The BD installer only detects Discord versions installed via
the .deb/.tar.gz from the Discord site
For AppStore/Flatpak/Snap versions, use betterdiscordctl

Cannot read property 'assets' of undefined


You are using a very outdated installer - please redownload from the BetterDiscord site


(this was an issue from over 3 years ago, it was fixed within days)

[Windows] EPERM: operation not permitted, open

The installer does not have the correct permissions for installing, can be resolved with a complete reinstall of Discord
